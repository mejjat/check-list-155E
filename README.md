# Revision Moteur 155E

Application web Flask pour la gestion des révisions de moteurs thermiques.

## Fonctionnalités

- Gestion des checklists de révision
- Génération de rapports PDF
- Suivi des statuts
- Interface responsive

## Prérequis

- Python 3.8+
- wkhtmltopdf
- Navigateur web moderne
